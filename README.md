 - 이 프로그램은 엑셀파일을 입력받아 3개의 폴더별(시기별 분류)로 각각의 텍스트파일로 출력하는 프로그램이다.

2.ex02_mearge1.py
 - f파일로 읽아서 g파일(text0)로 쓰는 프로그램

3.ex03.py

4.ex03_merge_true_all.py
